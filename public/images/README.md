## Image Optimization Tips

1. **Compression**: Use tools like TinyPNG, ImageOptim, or Squoosh to compress images while maintaining quality
2. **Format**: Use JPEG for photographs, WebP for better compression (with fallback)
3. **Responsive Images**: Consider creating multiple sizes (desktop, tablet, mobile) for better performance
4. **Alt Text**: All images already have descriptive alt text in the code for SEO and accessibility

## File Naming

- Use lowercase letters
- Use hyphens instead of spaces
- Keep filenames descriptive but concise

## Notes

- All images should be properly optimized for web use
- Ensure images are high quality but not excessively large (aim for < 500KB per image)
- Images should be properly licensed or owned by the author
